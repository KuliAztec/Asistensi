Kendaraan.java

Ringkasan singkat

Kelas dasar yang menyimpan atribut umum kendaraan, menyediakan constructor, method untuk menghitung sewa (dengan method overloading), serta variabel dan method static untuk statistik global.

Blok-blok kode

  1. Static variables protected static int totalKendaraan, protected static double totalPendapatan

    Digunakan untuk menghitung jumlah kendaraan yang dibuat dan mengakumulasi pendapatan dari transaksi sewa. Bersifat shared di semua instance.

  2. Instance variables protected String merk, protected int tahun, protected double hargaSewa

    Menyimpan atribut per kendaraan. protected memberi akses ke subclass dan kelas dalam paket yang sama.

  3. Constructor inisialisasi atribut dan totalKendaraan++

    Menggunakan this untuk membedakan parameter dan atribut. Panggilan constructor dari subclass lewat super(...) juga akan menambah counter global.

  4. Method overloading tiga versi hitungTotalSewa(...)

    1) hitungTotalSewa(int jumlahHari): perkalian sederhana dan menambahkan nilai ke totalPendapatan. 2) hitungTotalSewa(int jumlahHari, double diskon): memanggil versi 1 via this.hitungTotalSewa(jumlahHari) lalu mengaplikasikan diskon. 3) hitungTotalSewa(int jumlahHari, double diskon, double biayaAsuransi): memanggil versi 2 lalu menambahkan biaya asuransi.

    Catatan penting: Implementasi saat ini menambahkan ke totalPendapatan di versi 1. Karena versi 2/3 memanggil versi 1 terlebih dahulu, totalPendapatan bisa merekam jumlah sebelum diskon/asuransiini mungkin tidak sesuai maksud. Rekomendasi: hitung jumlah akhir terlebih dahulu, lalu tambahkan ke statistik, atau buat method transaksi terpisah.

  5. Metode presentasi tampilkanInfo()

    Menampilkan atribut dasar kendaraan ke konsol. Subclass dapat override dan memanggil super.tampilkanInfo() untuk mempertahankan output dasar.

  6. updateHarga(double)

    Mengubah harga sewa instance. Memanfaatkan this untuk mengakses atribut instance.

  7. Static methods getter & utilitas: getTotalKendaraan(), getTotalPendapatan(), tampilkanStatistik(), resetStatistik()

    Memungkinkan akses dan operasi pada statistik global tanpa membuat object. Perhatikan perlunya proteksi/otorisasi sebelum melakukan resetStatistik() pada aplikasi nyata.

Mobil.java

Ringkasan singkat

Subclass dari Kendaraan yang menambahkan atribut spesifik (jumlah pintu, jenis transmisi), override perilaku perhitungan sewa (markup 10%) dan menyediakan method tambahan terkait mobil.

Blok-blok kode

  1. Extends class Mobil extends Kendaraan

    Menandakan pewarisan. Mobil mewarisi atribut protected dan method dari Kendaraan.

  2. Static variable khusus private static int totalMobil

    Penghitung jumlah objek Mobil.

  3. Instance variable private int jumlahPintu, private String jenisTransmisi

    Atribut spesifik mobil; bersifat private sehingga hanya diakses melalui method class.

  4. Constructor memanggil super(...) dan menginisialisasi atribut sendiri

    super(merk, tahun, hargaSewa) memanggil constructor parent yang juga menambah totalKendaraan. Lalu totalMobil++ menambah penghitung mobil.

  5. Override hitungTotalSewa(int)

    Menambahkan markup 10%: return this.hargaSewa * jumlahHari * 1.1; menggunakan this untuk mengakses atribut protected parent.

  6. Overload hitungTotalSewa(int, boolean)

    Menambahkan biaya driver jika denganDriver==true. Memanggil versi local this.hitungTotalSewa(jumlahHari) sehingga override berlaku.

  7. Method yang memakai super hitungSewaStandard(int)

    Memanggil super.hitungTotalSewa(jumlahHari) untuk mendapatkan harga tanpa markup (mengakses method parent secara eksplisit).

  8. Override tampilkanInfo()

    Mencetak header khusus "MOBIL", memanggil super.tampilkanInfo() lalu menambahkan info jumlah pintu dan transmisi.

  9. Method perbandingan bandingkanHarga(int)

    Menampilkan perbandingan antara harga parent (standard) dan harga mobil (+10%). Berguna untuk demo this vs super.

  10. Static method getTotalMobil(), tampilkanInfoMobil()

    Info ringkas jumlah mobil terdaftar.

Motor.java

Ringkasan singkat

Subclass dari Kendaraan dengan atribut jenisMotor, overload method untuk menambahkan biaya helm, dan metode presentasi khusus.

Blok-blok kode

  1. Static variable khusus private static int totalMotor

    Penghitung jumlah objek Motor.

  2. Instance variable private String jenisMotor

    Menandai tipe motor (matic, sport, dll.).

  3. Constructor public Motor(...)

    Panggilan super(merk, tahun, hargaSewa) menginisialisasi atribut parent dan menambah totalKendaraan. Lalu totalMotor++ menambah penghitung motor.

  4. Method overloading hitungTotalSewa(int jumlahHari, int jumlahHelm)

    Memanggil super.hitungTotalSewa(jumlahHari) lalu menambahkan biaya helm (Rp 5.000 per helm per hari). Karena memanggil super, method ini tidak memanggil versi yang menambah pendapatan jika versi parent menambahkan; namun perhatikan bagaimana totalPendapatan dimutasi di parent pada panggilan yang berbeda.

  5. Override tampilkanInfo()

    Memanggil super.tampilkanInfo() dan menambahkan baris jenis motor.

  6. infoLengkap()

    Menampilkan detail lengkap motor memakai atribut protected dari parent (tahun, hargaSewa) untuk demo akses this dan akses protected.

  7. Static method getTotalMotor(), tampilkanInfoMotor()

    Menampilkan informasi jumlah motor terdaftar.

Main.java

Ringkasan singkat

Class berisi method main(String[] args) yang mendemonstrasikan penggunaan static members, pembuatan objek, pemanggilan method (this/super), method overloading, update field, dan simulasi transaksi.

Blok-blok kode & alur

  1. Banner dan text heading menampilkan judul aplikasi ke console.
  2. Demo STATIC sebelum membuat objek memanggil Kendaraan.getTotalKendaraan() dan method static class Mobil/Motor untuk menunjukkan nilai awal (biasanya 0).
  3. Pembuatan objek membuat dua mobil dan dua motor. Constructor subclass memanggil super(...) sehingga penghitung global dan per-kelas bertambah.
  4. Demo after-creation static menampilkan jumlah kendaraan, mobil, motor setelah pembuatan objek.
  5. Demo THIS dan SUPER
    • memanggil tampilkanInfo() (override + super)
    • menghitung sewa menggunakan override vs parent
    • menggunakan method perbandingan untuk menampilkan perbedaan harga
  6. Demo method overloading memanggil variasi hitungTotalSewa pada objek untuk menunjukkan overload behavior dan parameter tambahan (driver, helm).
  7. Demo updateHarga() menampilkan nilai, memanggil updateHarga(), lalu menampilkan nilai setelah update.
  8. Simulasi transaksi memanggil beberapa hitungTotalSewa dan menyimpan hasil ke variabel; panggilan ini juga mempengaruhi totalPendapatan berdasarkan implementasi di class parent.
  9. Tampilan statistik akhir memanggil Kendaraan.tampilkanStatistik() dan menampilkan jumlah per kategori.
